## Install
 Clone the repository and install the required dependencies by running
 ```sh
 git clone git@github.com:Chima1707/modus-create.git
 cd modus-create
 npm install
 ```

## Test
```sh
npm test
```

## Run 
```sh
npm start
```


The APIs can be found at
[APIs](http://localhost:8888/vehicles)
